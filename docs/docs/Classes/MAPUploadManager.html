MAPUploadManager

@interface MAPUploadManager

The MAPUploadManager class handles both image processing and uploading.

Properties
  • delegate

    The delegate of the upload manager object.

  • testUpload

    Set this flag to YES if you want to test your upload. It uploads to a test server instead of the production server. Images on this server is not accessible and will be deleted.

    Default is NO.

  • deleteAfterUpload

    Set this flag to NO if you want to keep your local images efter they have been uploaded. This flag is only used if testUpload is set to YES. When uploading to the production servers images are always deleted to avoid duplicates.

    Default is YES.

  • allowsCellularAccess

    Set this flag to YES if you want to allow uploads via cellular network.

    Default is NO.

Creating an Upload Manager
  • +sharedManager

    Returns the shared upload manager object.

    Return Value

    The shared upload manager.

Starting and stopping image processing
  • -processSequences:

    Starts to process the sequences in the array.

  • -stopProcessing

    Stops the image processing.

Starting and stopping image uploading
  • -processAndUploadSequences:

    Starts to process and upload the sequences in the array.

  • -stopUpload

    Stops the image processing and upload.

Getting the current status of the Upload Manager
  • -getStatus

    Returns the current status of the Upload Manager.

    Return Value

    The current status of the Upload Manager.
