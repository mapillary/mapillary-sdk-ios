MAPImage

@interface MAPImage

The MAPImage class encapsulates the information about an image.

Properties
  • captureDate

    The date the image was captured.

  • imagePath

    The path to the image file on disk.

  • location

    The location the image was captured.

  • author

    The user that captured the image.

Initializers
  • -init

    Creates an image object with the currently logged in user and the current date.

  • -initWithPath:

    Creates an image object with the currently logged in user and the current date. If a file exists at the path and a thumbnail is missing, one is created.

    Parameters

    path

    The path to the image file.

Image loading
  • -loadImage

    Loads the full-size image file as an UIImage.

    Return Value

    The loaded image.

  • -loadThumbnailImage

    Loads a low resulution version of the image file as an UIImage.

    Return Value

    The loaded image.

Misc
  • -thumbPath

    Returns the file path to the thumbnail.

    Return Value

    The file path to the thumbnail.

  • -isLocked

    Returns the lock status. When a MAPImage is scheduled for processing or uploading, it is locked until the operation is complete. When a MAPImage is locked it cannot be modified or deleted.

    Return Value

    If the file is locked not not.
