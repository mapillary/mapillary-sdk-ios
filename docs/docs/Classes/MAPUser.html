MAPUser

@interface MAPUser

The MAPUser class encapsulates the information about a user.

Properties
  • userName

    The user’s username. It can be changed by the user.

  • userKey

    The unique key to identify the user. The userKey never changes.

  • userEmail

    The user’s email adress. It can be changed by the user. It will only be non-nil for the currently signed in user.

  • accessToken

    The user’s access token. It will only be non-nil for the currently signed in user.

    See
