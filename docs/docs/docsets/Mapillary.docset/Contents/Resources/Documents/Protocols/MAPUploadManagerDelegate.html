MAPUploadManagerDelegate

@protocol MAPUploadManagerDelegate

This protocol is used for notifying the delegate what’s happening during image processing and upload.

  • -imageProcessed:image:uploadStatus:

    Delegate method for when an image is processed.

    Parameters

    uploadManager

    The upload manager object that is processing the image.
    image

    The image that has been processed.
    uploadStatus

    The current status of the upload manager.
  • -processingFinished:uploadStatus:

    Delegate method for when all images have been processed.

    Parameters

    uploadManager

    The upload manager object that is processing the image.
    uploadStatus

    The current status of the upload manager.
  • -processingStopped:uploadStatus:

    Delegate method for when image processing has was stopped by calling stopProcessing.

    Parameters

    uploadManager

    The upload manager object that is processing the image.
    uploadStatus

    The current status of the upload manager.
  • -imageUploaded:image:uploadStatus:

    Delegate method for when an image was sucessfully uploaded.

    Parameters

    uploadManager

    The upload manager object that is uploading the image.
    image

    The image that has been uploaded.
    uploadStatus

    The current status of the upload manager.
  • -imageFailed:image:uploadStatus:error:

    Delegate method for when an image failed to upload.

    Parameters

    uploadManager

    The upload manager object that is uploading the image.
    image

    The image that failed to upload.
    uploadStatus

    The current status of the upload manager.
    error

    The error explaining what went wrong.
  • -uploadedData:bytesSent:uploadStatus:

    Delegate method for when a chunk of data has been uploaded.

    Parameters

    uploadManager

    The upload manager object that is uploading the images.
    bytesSent

    The number of bytes sent since the last invocation of this method.
    uploadStatus

    The current status of the upload manager.
  • -uploadFinished:uploadStatus:

    Delegate method for when all images have been uploaded.

    Parameters

    uploadManager

    The upload manager object that is uploading the images.
    uploadStatus

    The current status of the upload manager.
  • -uploadStopped:uploadStatus:

    Delegate method for when image uploading has was stopped by calling stopUpload.

    Parameters

    uploadManager

    The upload manager object that is uploading the images.
    uploadStatus

    The current status of the upload manager.
